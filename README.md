## 🚀 Getting Started

### Prerequisites
- Node.js 18+ 
- npm or yarn
- Git

### Installation

```bash
# Clone the repository
git clone https://github.com/farkhanmaul/j7portofolio.git

# Navigate to project directory
cd j7portofolio

# Install dependencies
npm install

# Start development server
npm run dev
```

### Deployment

```bash
# Build for production
npm run build

# Deploy to GitHub Pages
npm run deploy
```
